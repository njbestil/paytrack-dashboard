# PayTrack Dashboard

PayTrack Dashboard is a simple POS transaction dashboard built as a portfolio project. The goal of this project is to demonstrate practical full-stack development skills using ReactJS, TypeScript, Tailwind CSS, Node.js, Express.js, and a REST API backend.

This project focuses on building a clean, job-relevant dashboard for managing and viewing POS payment transactions.

## Features

* Login page UI
* Dashboard summary cards
* Transaction list
* Search transactions
* Filter transactions by payment status
* Transaction details page
* Add test transaction form
* Basic user role display
* Responsive dashboard layout
* Mock transaction data for frontend development

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/your-username/paytrack-dashboard.git
```

### 2. Go to the project folder

```bash
cd paytrack-dashboard
```

### 3. Install dependencies

```bash
npm install
```

### 4. Run the development server

```bash
npm run dev
```
